# game_L905_target_hunt

ターゲットハントゲーム - 動き回るターゲットを追いかけるクリックゲーム

## 🎮 ゲーム概要

制限時間30秒間で、ランダムに移動するターゲットをクリックして倒すゲームです。ターゲットは0.7秒ごとに位置を変えるため、集中力と反射神経が試されます。連続で命中するとコンボが記録されます。

## 🚀 デモ

[GitHub Pagesでプレイ](https://kmh-no3.github.io/game_L905_target_hunt/)

## 📋 機能

- **タイムアタック**: 30秒間で何回命中できるか挑戦
- **コンボシステム**: 連続で命中するとコンボが記録される
- **動的ターゲット**: 0.7秒ごとにランダムに移動するターゲット
- **リアルタイム統計**: 残り時間、命中数、コンボ数をリアルタイム表示

## 🛠️ 技術スタック

- **TypeScript**: 型安全なJavaScript
- **Vite**: 高速なビルドツール
- **Vanilla CSS**: カスタムスタイリング

## 📦 セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

ビルド成果物は `dist/` フォルダに生成されます。

## 🎯 使い方

1. 「スタート」ボタンをクリックしてゲーム開始
2. フィールド内に表示される赤いターゲットをクリック
3. ターゲットをクリックすると、新しい位置に移動します
4. 30秒間でできるだけ多くのターゲットを倒しましょう
5. 連続で命中するとコンボが記録されます
6. フィールドからマウスが離れるとコンボがリセットされます
